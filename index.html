Welcome to C# Programming Introduction!

C# is a modern, object-oriented programming language that is widely used for building Windows applications, web applications, and mobile apps.

Getting Started with C#

To get started with C#, you'll need to install Visual Studio, a popular integrated development environment (IDE) for C# development.

  • Download and install Visual Studio from the official website.
  • Create a new C# project in Visual Studio.
  • Write your first C# program using the `Console.WriteLine` method.
  • Download Visual Studio

Basics of C#

Variables

In C#, a variable is a named storage location that holds a value.

int x = 5;

char myChar = 'A';

float myFloat = 3.14f

public static void (your method or local function);

double myDouble = 3.14159;

Data Types

C# has several built-in data types, including integers, strings, and booleans.

string name = "John";

To use boolean we need using true or false. Example:

int y = 10;

0 = true ; 1 = false;

Control Structures

Conditional (if/else): executes a block of code if a condition is true

Loops (while, for, foreach): executes a block of code multiple times

Switch: executes a block of code based on a value

Methods

Are blocks of code that can be called multiple times, Can take parameters and return values

switch (fullName)

}

Classes and Objects

Classes: are blueprints for creating objects

Objects: are instances of classes

Properties: are characteristics of an object

Methods: are actions an object can perform
